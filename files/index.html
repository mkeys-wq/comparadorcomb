preços atualizados diariamente · DGEG / ERSE
Esta demonstração usa dados de exemplo para ilustrar a experiência. Os preços reais e atualizados diariamente encontram-se disponíveis em precoscombustiveis.dgeg.gov.pt (DGEG/ERSE). Liga esta app a esses dados através de um pequeno servidor intermédio, já que a API pública da DGEG não pode ser acedida diretamente a partir do browser.
Os postos de carregamento elétrico apresentados são inventados para demonstração — não vêm de nenhum registo público real. A DGEG não publica dados de carregamento elétrico; para dados reais e atualizados seria preciso ligar a app à rede MOBI.E (rede pública portuguesa) ou a um agregador como Electromaps, também através de um servidor intermédio.
